A beginner-friendly inventory management system in Python, with CSV file-handling.
FEATURES:
- View Inventory
- Add Items
- Remove Items
- Update Item details

Example csv format:

items = [
["items", "Quantity", "Price"],
["Pen", 50, 10.5],
["Notebook", 20, 45.0],
["Eraser", 100, 5.25]
]

I learned:
1. how to manipulate, save, and update csv files.
2. Create modular code.
